# RAN - Repository for Automation and Networking
Central repository for home automation, networking, and data processing projects.

## Major Projects
```
RAN/
├── CRAP/          # Collection of Reusable Automated Pipelines [→ data processing]
├── Services/      # Docker services and infrastructure [→ active services]
├── Linux/         # System initialization scripts
├── Win11/         # Windows utilities and scripts
├── docs/          # Documentation and plans
├── utils/         # Shared utilities (gppu.py, etc.)
└── archive/       # Deprecated code and old versions
```
